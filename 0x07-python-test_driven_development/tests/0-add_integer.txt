Using ``add_integer``
----------------------

This module defines an integer addition function ``add_integer(a, b=98)``.
First import ``add_integer`` from ``0-add_integer.``

>>> from 0-add_integer import add_integer

Now use it:

>>> add_integer(2, 3)
	5

	Works for float
>>> add_integer(2.0, 3.0)
	5.0
